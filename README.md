## Overview
A big data classification project to predict whether a mushroom is 
edible or poisonous based on physical characteristics.

## Dataset
- Source: Kaggle
- Size: 50,000+ rows
- Label: Edible / Non-edible (binary classification)

## Methodology
- Compared multiple classification algorithms
- Selected GradientBoostTree as final model based on performance benchmarking

## Tech Stack
- Python
- Apache Spark
- Google Colab

## How to Run
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook
2. Install dependencies: `pip install pyspark`
3. Run all cells sequentially
